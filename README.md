در فایل `api.js` یک سیستم کش هوشمند وجود دارد:

- اگر داده قبلاً ذخیره شده باشد و کمتر از 24 ساعت گذشته باشد → از localStorage خوانده می‌شود
- در غیر این صورت → درخواست جدید به API ارسال می‌شود
- اگر داده نامعتبر باشد → نادیده گرفته می‌شود
